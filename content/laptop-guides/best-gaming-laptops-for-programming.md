Programming laptops have a different priority list from pure gaming rigs: lots of RAM (and the ability to add more), a strong multi-core CPU for compiles and containers, a comfortable keyboard, solid build and decent battery for working away from a desk. A capable GPU is a bonus for local ML or game dev, but memory and serviceability come first.

The Framework Laptop 16 is the standout for developers who value longevity — its 10/10 repairability, swappable modules and 96 GB RAM ceiling mean it can be maintained and upgraded for years. The Lenovo Legion Pro 5 and Pro 7i bring genuinely fast multi-threaded CPUs with upgradeable RAM and good aluminum builds, while the XMG Core 16 pushes memory to a huge 128 GB. The ASUS TUF A18 and F16 are the value picks, both upgradeable and the A18 unusually clean on thermals.

How we chose: we required user-upgradeable RAM and weighted CPU core count, build quality, battery and cooling over peak GPU output.

A caveat: the most portable, premium machines elsewhere in our coverage — the Razer and Zephyrus lines — use soldered RAM, which is why they are absent here; for a dev machine you want to add memory later. Also note the Framework carries a clear price premium and a lower-wattage GPU, so you buy it for the philosophy and repairability, not raw frame rates.
